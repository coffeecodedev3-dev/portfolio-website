Modern Expense Tracker Dashboard

Project Description

A sleek, responsive personal finance management tool built with HTML, Tailwind CSS, and Vanilla JavaScript. This dashboard provides a real-time overview of your financial health, allowing you to track income, expenses, and overall balance with ease.

Features

  • Dashboard Overview: Real-time tracking of Total Balance, Total Income, and Total Expenses.
  • Transaction Management: Easily add new income or expenses with descriptions and categories.
  • Dynamic History: View a styled table of recent transactions with color-coded types (Income/Expense).
  • Responsive Design: Fully functional on mobile, tablet, and desktop screens thanks to a modern Tailwind CSS layout.
  • Data Integrity: Smart balance calculations that handle deletions accurately using data-attributes.

Technologies Used

  • HTML5
  • Tailwind CSS (via CDN)
  • Vanilla JavaScript (ES6+)
  • Local Storage / Data Attributes for state management

How to Run

  1. Clone the repository from GitHub.
  2. Open the index.html file in any modern web browser.
  3. Alternatively, visit the Live Demo.
