# CoffeeCodeDev Portfolio Website

A modern, responsive, dark-themed portfolio website designed to showcase projects in Java, Web Development, and Automation.

## 🚀 Features

- **Modern Dark Aesthetic:** Utilizes a professional dark-mode palette (#2B2A2A) with vibrant accents.
- **Categorized Project Gallery:** Projects are organized into sections: Java, Websites, and Automation.
- **Difficulty Badges:** Each project is labeled with its difficulty level (e.g., Easy, Intermediate) to guide visitors.
- **Responsive Design:** Fully optimized for mobile and desktop browsing.
- **Detailed Project Pages:** Individual pages for deep-dives into project architecture, features, and instructions.

## 🛠️ Built With

- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling using CSS Variables and modern Flexbox/Grid layouts.
- **Git:** Version control and deployment.

## 📂 Project Structure

- `Home.html`: The landing page and project gallery.
- `Style.css`: Central stylesheet for the entire site.
- `Navigation/`: Contains secondary pages like "About Us".
- `portfolio/`: Detailed pages for individual projects (e.g., Core Banking Application, Lead Tracker).
- `assets/`: Image assets and profile photos.

## 🖥️ Getting Started

To view the website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/coffeecodedev3-dev/portfolio-website.git
   ```
2. Open `Home.html` in your preferred web browser.
