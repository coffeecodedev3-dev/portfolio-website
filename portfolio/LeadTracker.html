Lead Tracker Application

Project Description

The Lead Tracker Application is a robust, console-based Lead Management System built using Java. This project follows a clean 3-tier architecture, ensuring a clear separation of concerns between data management, business logic, and the presentation layer.

Key Features

  • Lead Management: Comprehensive tracking of leads through various status transitions such as New, Contacted, Won, and Lost.
  • Data Integrity: Implements advanced custom exception handling and robust input validation to ensure reliable data processing.
  • Analytics & Reporting: Leverages the powerful Java Stream API for efficient revenue tracking and real-time status reporting.
  • 3-Tier Architecture: Modular design for better maintainability and scalability.

Technologies Used

  • Java 17+
  • Maven (Build Tool)
  • JUnit 5 (Testing Framework)
  • Java Stream API
  • Object-Oriented Design: Encapsulation, Separation of Concerns.

How to Run

  1. Clone the repository from GitHub.
  2. Build the project using Maven: ./mvnw clean install
  3. Run the application: ./mvnw exec:java -Dexec.mainClass="com.aryapanda.LeadTracker.LeadTrackerRunner"
  4. Run tests: ./mvnw test
