Banking Application

Project Description

Java Banking Application is a console-based banking application written in pure Java. This project demonstrates core OOP (Object-Oriented Programming) principles, Exception Handling, Custom exception classes, and robust input validation.

Features

  • Account Creation: Users can register a new account with a name, password, and initial deposit.
  • User Login: Secure login for existing users.
  • Core Banking Operations:
    • Check Balance
    • Deposit Funds
    • Withdraw Funds (with insufficient funds check)
    • Transfer Funds (to another account)
  • Error Handling: The app uses custom exceptions for robust error handling.
  • Input Validation: The app features "uncrashable" input methods that loop until valid data is entered.

Technologies Used

  • Core Java
  • Object-Oriented Principles (OOP): Encapsulation, Inheritance, Polymorphism.
  • Data Structures: HashMap for storing and retrieving account data.
  • Exception Handling: Custom exceptions for business logic.

How to Run

  1. Clone the repository from GitHub.
  2. Open the project in an IDE (like VS Code or Eclipse).
  3. Run the Main.java file located in src/com/bankapp/
